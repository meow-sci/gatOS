gatOS uses [Alpine Linux](https://alpinelinux.org/) as its upstream distro.

Information you find here will be generic Linux oriented (preferable) or Alpine-specific where needed.

## Installing pre-packaged software

gatOS uses Alpine as its upstream Linux distribution which comes with a first-party package manager **apk** which is fast and efficient.

::NOTE:: afa

## Install language runtimes



### Python

```bash
apk add python3
```
